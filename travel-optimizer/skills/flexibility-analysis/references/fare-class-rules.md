# Fare Class Rules by Airline

## Understanding Fare Classes

Airlines use single-letter codes to define ticket types. Each code determines pricing, change/cancel rules, mileage earning, and upgrade eligibility.

## General Rules

1. **24-Hour Free Cancellation**: US DOT requires airlines to offer free cancellation within 24 hours of booking for flights to/from US
2. **EU261 Protection**: For flights from EU airports, compensation for delays/cancellations regardless of fare class
3. **No-show**: Almost always forfeits entire ticket value, even on flexible fares (must cancel BEFORE departure)
4. **Credit voucher validity**: Typically 12 months from issue date
5. **Name changes**: Generally not permitted (exception: some corporate accounts)
