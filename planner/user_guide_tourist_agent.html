📌 Tổng quan

Tourist Agent là bộ công cụ tối ưu hóa tự động quá trình lên lịch trình và ngân sách du lịch, kết hợp sức mạnh của Travel Optimizer AI.

Dành cho ai: Gia đình, nhóm bạn, hoặc bất kỳ ai muốn tự lập lịch trình đi Nhật chuyên nghiệp với chi phí được tính toán minh bạch.

Tiện ích vượt trội: Không cần cài đặt bất cứ ứng dụng nào, không cần API Key, không đòi hỏi cấu hình máy chủ. Chỉ cần click đúp vào các tệp HTML (chạy offline trên trình duyệt web) là dùng được ngay!

📖 Hướng dẫn từng Phase

Phase 1 — Prompt Assistant
📄 File: tourist_agent_prompt_assistant_phase1_20260529.html

Làm gì: Điền thông tin cơ bản về chuyến đi (số người, ngân sách, số ngày, nhịp độ). Hệ thống sẽ phân tích để đưa ra 5 câu hỏi làm rõ nhu cầu và một đoạn Prompt "sạch".

Output: Câu lệnh Prompt chuẩn SEO dùng để copy-paste thẳng vào các công cụ AI như ChatGPT, Gemini, hay Claude nhằm lấy được một lịch trình hoàn chỉnh.

→ Mở Phase 1 (mở tab mới)
Phase 2 — Lịch trình nháp
📄 File: tourist-agent-phase2.html

Làm gì: AI sinh ra lịch trình gợi ý dựa trên đoạn prompt bạn đã copy từ Phase 1. Hệ thống sẽ kết xuất tự động một lịch trình nháp dạng bảng, thể hiện rõ các điểm đến phân chia theo buổi sáng, chiều, tối.

Output: Bảng lịch trình nháp theo ngày và theo nhịp độ (Thư thả, Cân bằng, Tối ưu nhiều điểm).

→ Mở Phase 2 (mở tab mới)
Phase 3 — So sánh vé máy bay
📄 File: tourist-agent-phase3.html

Làm gì: Tự động tạo prompt để yêu cầu Travel Optimizer tìm các chuyến bay. Bạn copy JSON do AI tạo ra và dán ngược lại công cụ.

Lưu ý: Dùng chế độ AI-Knowledge Mode không đòi hỏi API Key thật nên có thể dễ dàng chạy mô phỏng kết quả cho bài lab giảng dạy.

Output: Bảng 3 phương án vé chi tiết bao gồm giá trị thực và phân bổ ngân sách du lịch tổng thể.

→ Mở Phase 3 (mở tab mới)
Phase 4 — Verification Gate (Cổng kiểm chứng)
📄 File: tourist-agent-phase4.html

Làm gì: Hệ thống tự sinh Prompt để kiểm tra rủi ro (verification). Bạn copy JSON trả về từ AI và dán vào, sau đó nhấn "Render Checklist".

Output: Checklist 5 hạng mục của từng ngày hiển thị trạng thái (confirmed, assumption, needs_verification) cùng danh sách deals khuyến mãi khả dụng.

→ Mở Phase 4 (mở tab mới)
Phase 5 — Kế hoạch cuối
📄 File: tourist-agent-phase5.html

Làm gì: Tất cả quá trình từ Phase 1 - 4 hội tụ về đây. Tự động xuất 3 Artifacts cực kỳ hữu ích chỉ qua một click chuột.

Output gồm 3 Tab:

  • Tab 1: Lịch trình cuối: Bảng tổng hợp theo từng ngày (accordion), chi tiết đến từng địa điểm và có summary vé máy bay nổi bật.
  • Tab 2: Email / Zalo message: Soạn sẵn văn bản thuần, sẵn sàng gửi. Tự động đi kèm deal/mã giảm giá hợp lệ.
  • Tab 3: Todo List: Danh sách việc cần làm có thể tick, phân rõ Nhóm A (Cần làm ngay), Nhóm B (Sau khi chốt vé) và Nhóm C (Việc cần kiểm chứng thêm từ Verification Gate).
→ Mở Phase 5 (mở tab mới)

Hai Chế Độ Hoạt Động:

  • AI-Knowledge Mode: Sử dụng kiến thức AI để ước lượng giá. Hoàn toàn miễn phí, không cần API, chạy được lập tức. Đủ tốt cho phần lớn use cases.
  • API-Enhanced Mode: Kết nối trực tiếp với Kiwi hoặc Amadeus bằng Python để lấy giá vé thời gian thực.

❓ Câu Hỏi Thường Gặp (FAQ)

1. Tôi không biết code, dùng được không?
Có, rất đơn giản! Các tệp công cụ là HTML thuần, không cần cài đặt. Bạn chỉ cần double-click (nhấp đúp chuột) vào file để mở trên Chrome, Edge hoặc Safari và bắt đầu sử dụng.
2. Giá vé trong tool có chính xác 100% không?
Tất cả dữ liệu về giá vé là ước tính (estimated) để lên ngân sách dựa trên kiến thức của AI. Bạn luôn cần truy cập Google Flights, Skyscanner, hoặc website chính thức để kiểm chứng và đặt chỗ thật.
3. Phase nào bắt buộc phải làm?
Phase 1 và Phase 2 là bắt buộc để tạo ra kế hoạch cơ bản. Phase 3, 4, và 5 tùy chọn nhưng rất khuyến khích thực hiện để bạn có bản kế hoạch hoàn hảo nhất.
4. Travel Optimizer cần cài đặt gì không?
Nếu sử dụng AI-Knowledge Mode, bạn không cần cài gì cả. Mở file HTML lên và dùng ngay. Nếu muốn dùng API Mode, bạn cần Python + API key (chi tiết xem tại README của Travel Optimizer).
5. Dùng AI nào (Gemini, ChatGPT, Claude) tốt nhất?
Bộ công cụ cung cấp Prompt tương thích tốt với mọi LLM lớn. Bạn có thể sử dụng Gemini 1.5, ChatGPT (GPT-4o), hoặc Claude 3.5 Sonnet đều sinh ra kết quả JSON cực kỳ chất lượng để dán vào công cụ.

⚠ Khuyến cáo quan trọng: Tất cả thông tin giá vé, giờ mở cửa, và lịch trình trong tool này là ước tính do AI tạo ra. Luôn kiểm chứng thực tế trên Google Flights, Booking.com, và trang web chính thức trước khi đặt dịch vụ.
